# 🛡️ Pass-Brute v1.0 
**A Lightweight Password Security Testing Tool for iSH & Termux**

![License](https://img.shields.io/badge/License-MIT-green.svg)
![Platform](https://img.shields.io/badge/Platform-iSH%20%7C%20Termux-blue.svg)
![Language](https://img.shields.io/badge/Language-Shell-orange.svg)

Pass-Brute is an automated script designed for **Cybersecurity Researchers** and **Students** to test the strength of passwords using dictionary attacks. It is optimized to run smoothly on mobile terminal environments like iSH Shell (iOS) and Termux (Android).

## 📥 Installation

```bash
# Update your packages
apk update && apk upgrade

# Install git
apk add git

# Clone the repository
git clone [https://github.com/kinh17631-star/Pass-Brute.git](https://github.com/kinh17631-star/Pass-Brute.git)

# Enter the directory
cd Pass-Brute

# Give execution permission
chmod +x Pass-Brute.sh

# Run the tool
./Pass-Brute.sh
